Workflow

From upload to
verified model in minutes.

01

Upload IFC models

Upload .ifc or .ifczip files to the Documents module. Files are stored persistently in R2 cloud storage.

02

Configure checks

Select models to analyze, configure rules for the Checking module, and define clash groups per project.

03

Run analysis

Execute rule checks and clash detection in seconds. Results include element details, severity, and GlobalIds.

04

Export & resolve

Export results as BCF 2.1 or Excel. Track issues with status and priority until resolved.
